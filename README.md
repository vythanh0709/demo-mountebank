# Demo for Mountebank mocking npm package
Link: https://www.mbtest.org/
## mb
Project demo for mountebank\
Install npm package:
```
npm i
```
Run with json config
```
npm start
```
Run with blank project, config with REST api
```
npm run mb
```
## server1-ag
Basic Angular project.\
